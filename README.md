# Slivana School Website

A simple and great website for Slivana School, built with HTML, CSS, and JavaScript.

## Features

- Home page with hero banner, latest news, featured podcast and video, quick links
- About page with school information
- News page with searchable news articles
- Podcasts page with embedded episodes
- Videos page with school videos
- Gallery, Events, Admissions, Staff, Contact pages
- Mobile-friendly responsive design
- WhatsApp contact button
- Contact form and social links

## Structure

- `index.html` - Home page
- `about.html` - About the school
- `news.html` - News and announcements
- `podcasts.html` - Podcast episodes
- `videos.html` - School videos
- `gallery.html` - Photo gallery
- `events.html` - Upcoming events
- `admissions.html` - Admission information
- `staff.html` - Staff directory
- `contact.html` - Contact form and details
- `css/` - Stylesheets
- `js/` - JavaScript files
- `data/` - JSON data files
- `assets/` - Images and other assets

## Running the Website

To run the website locally:

1. Ensure you have Python installed
2. Navigate to the project directory
3. Run `python3 -m http.server 8000`
4. Open `http://localhost:8000` in your browser
